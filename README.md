# Ant Research Database
## 開発の目標
長野県の多様な**植生環境（森林、草地、里山、都市緑地）**に対応するアリ類群集のデータを集約・可視化する。
地表面環境の変化に敏感なアリ類を**「植生と土壌環境の健全性を測る生物指標」**と位置づけ、自然保護と土地利用管理を支援します。

### なんでアリ類か
* **植生環境の監査:** 立地環境や管理頻度（草地、農地、二次林、人工林等）ごとに組成が異なる
* **環境変化の可視化:** 土地利用の変化や植生の遷移が、生態系に与える影響をモニタリング
* **地域生態系の保全:** 植生を守る基礎データを提供
